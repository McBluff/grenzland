# Grenzland 1823 — Räuber und Polizist

Ein digitales Brettspiel über Schmuggler im Bündner Prättigau, anno 1823.

## Online spielen

Die HTML-Version dieses Spiels ist über GitHub Pages online verfügbar.
Wenn du den Link zur HTML-Datei auf zwei Geräten gleichzeitig öffnest,
könnt ihr im Online-Multiplayer gegeneinander spielen.

## Spielmodi

- **Hot-Seat:** zwei Spieler an einem Computer
- **Gegen Computer:** Solo gegen die KI
- **Online-Multiplayer:** zwei Geräte (z.B. MacBook und iPad)

## Features

- Hexagonales Spielfeld im Brettspiel-Stil
- Aquarell-Medaillons für alle Figuren
- Stealth-Mechanik (Räuber sind für Polizist nur auf Felsen sichtbar)
- 15 verschiedene Wetter- und Ereigniskarten
- Drag-and-Drop-Bewegung
- KI-Gegner mit unscharfer Sicht
